# 💰 Finance Dashboard UI

A modern and responsive finance dashboard built to visualize and manage financial data effectively. This project demonstrates frontend development skills including UI design, component structuring, and state management.

---

## 🚀 Features

### 📊 Dashboard Overview

* Summary cards:

  * Total Balance
  * Income
  * Expenses
* Line chart showing balance trends over time
* Pie chart for spending category breakdown

### 📋 Transactions Section

* Transaction table with:

  * Date
  * Amount
  * Category
  * Type (Income / Expense)
* Features:

  * Search functionality
  * Filtering by transaction type
  * Sorting support

### 🔐 Role-Based UI

* Role switcher: **Admin / Viewer**
* Admin:

  * Add, edit, delete transactions
* Viewer:

  * Read-only access (buttons disabled)

### 💡 Insights Section

* Highest spending category
* Monthly comparison
* Savings summary

---

## 🧠 Tech Stack

* **React (Vite)**
* **Tailwind CSS**
* **Recharts** (for data visualization)
* **React Context API** (state management)

---

## 🎨 UI/UX Highlights

* Clean and modern fintech-style design
* Fully responsive (mobile + desktop)
* Smooth transitions and hover effects
* Consistent spacing and typography
* Dark mode support

## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone <your-repo-link>
cd zorvyn-dashboard
```

2. Install dependencies:

```bash
npm install
```

3. Run the project:

```bash
npm run dev
```

4. Open in browser:

```
http://localhost:5173
```

## 🏗️ Approach

* Designed a modular component structure for scalability
* Used React Context to manage:

  * Transactions data
  * Filters
  * User role
* Implemented reusable UI components for consistency
* Focused on simplicity, clarity, and usability over complexity
